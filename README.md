# Healthcare-Analytics-Dashboard

1. Project Title / Headline
Healthcare Analytics Dashboard

An interactive Power BI dashboard designed to analyze healthcare operational and specialty-level data, helping users identify trends, performance metrics, and patient-related insights across different medical specialties.

2. Short Description / Purpose

The Healthcare Analytics Dashboard is a dynamic and visually engaging Power BI report created to simplify healthcare data analysis through interactive visualizations and KPI tracking. The dashboard helps users monitor specialty performance, analyze operational trends, and gain actionable insights for data-driven healthcare decision-making.

This dashboard is intended for healthcare administrators, analysts, operational teams, and decision-makers who need a centralized view of healthcare performance and specialty-level analytics.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main platform used for dashboard development and interactive reporting.

📂 Power Query – Used for data cleaning, transformation, and preparation.

🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic filtering logic.

🔗 Data Modeling – Relationships built between healthcare and specialty datasets for seamless cross-filtering and drill-down analysis.

📁 File Format – .pbix for development and dashboard creation.

4. Data Source
Source:

Healthcare operational and specialty-level datasets containing information related to medical specialties, healthcare services, and performance metrics.

The dataset includes:

Specialty groups
Healthcare operational metrics
Department/specialty-level performance indicators
Comparative analytics for healthcare reporting

5. Features / Highlights
• Business Problem

Healthcare organizations often manage large amounts of operational and specialty-level data, making it difficult to quickly identify trends, performance gaps, and high-performing specialties.

Without a centralized analytics solution, stakeholders struggle to:

Compare specialty performance
Monitor operational efficiency
Identify trends and outliers
Generate quick insights for decision-making

• Goal of the Dashboard

The goal of this dashboard is to provide an interactive healthcare analytics solution that:

Simplifies healthcare data analysis
Enables specialty-level performance monitoring
Supports operational and strategic healthcare decisions
Improves visibility into healthcare trends and KPIs
